Initialize a python environment and install the requirements:

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Please work through branches and pull requests. No direct commits onto the main branch.

Currently our work is based on Daunhawer's paper and their supporting [Github repo](https://github.com/imantdaunhawer/multimodal-contrastive-learning).
